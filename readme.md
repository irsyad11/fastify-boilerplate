## Getting started
this is a Starter KIT that creater made, to help future develepor 
to create an new backend application using javascript with framework Fastify
this utilize another package like :
1. JWT 
2. Bcrypt
3. Winston
4. axios

## installing
if u using npm, simply use :
npm install

if u using yarn :
yarn install
